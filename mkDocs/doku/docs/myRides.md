# My Ride Offers

The **My Ride Offers** page provides you with an overview of all the trips you have already offered. It is clearly structured so you can see all relevant information at a glance.

## Page Structure:

### Map (Top):
At the top of the page, there is an interactive map displaying all locations where you have offered rides. Each marker represents a ride offer.

### Overview (Below):
- **Title:** In the center, you will see the title **My Ride Offers**.
- **Earnings (Left):**  
  To the left of the title, the total amount (in euros) you have earned by offering rides is displayed. This shows how much additional income you have generated by taking passengers with you.
- **Saved Trips (Right):**  
  On the right side, you can see how many trips have been saved by sharing rides. This helps you understand the positive impact of your carpooling on traffic and the environment.
- **Create New Ride Offer:**  
  To the right of the saved trips, there is a button that allows you to create a new ride offer. [This is explained in more detail below.](#create-ride)
- **Ride Overview (Below):**  
  Below the overview, you will find a table listing all the rides you have created, allowing you to view detailed information about each offer.

### Interactive Buttons (Next to Each Ride):

- **Search Function:**  
  A button allows you to highlight a specific ride. The corresponding marker on the map will be shown in a different color for easy visual identification.

- **Edit:**  
  Another button allows you to edit an existing offer. This lets you make changes if your plans change or if you want to update ride details.

- **Cancel:**  
  A button to cancel an offer if you no longer wish to provide the ride.

!!! warning "Status-Dependent Buttons"
    Depending on the status of a ride (e.g., whether it has already been accepted or canceled), certain buttons may be hidden or disabled, as they are no longer needed. This helps keep the interface clean and easy to use.

## Create a Ride {#create-ride}

The **Create Ride** page allows you to quickly and easily create a new carpool offer. Below is a detailed overview of the page structure and its features:

### Left Side – Form:
On the left side, there is a form where you can enter all the important details for your ride offer. This includes information such as departure location, destination, time, and other relevant trip details.

### Error Messages:
If you fill in a field incorrectly or leave something out, the affected field will be highlighted in red, and an error message will appear below it explaining the issue. This helps you quickly identify and fix mistakes.

### Required Fields:
Not all fields need to be filled out, but some are marked with an asterisk (*). These are required fields for creating an offer. It is recommended to provide as much information as possible to help other users better understand where and when the ride takes place.

### Buttons:
- **Cancel:**  
  A button to abort the process if you do not want to create the ride.

- **Create:**  
  A button to create the ride offer once all required data has been entered correctly.

### Right Side – Map:
On the right side, there is an interactive map that visually represents your entered data. Once you fill in the required fields, a marker will appear on the map showing the specified departure and/or destination.

!!! bug "Troubleshooting"
    If the marker appears far from the intended location, please check whether the address information was entered correctly. Typos or incomplete details can sometimes lead to incorrect placement. If you notice any inconsistencies, correct the address data and verify that the marker is now positioned correctly.
