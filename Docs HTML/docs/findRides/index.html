    Find Ride Offers

    The Find Ride Offers page is designed to help users find a suitable carpool opportunity. The page is structured as follows:

    Map

    At the center of the page is a large map. The vocational school Rodewisch is marked with a red marker, while available ride offers are displayed as green markers.

    Filter Search

    If you want to narrow down your search, you can click a button in the top-left corner. There, you’ll find a slider that allows you to adjust the radius for displayed rides. The selected radius is highlighted on the map so you can clearly see which offers fall within your desired area.

    Select an Offer

    When you find a suitable offer, you can simply click on its marker. This opens a panel on the right side containing all important information about the offer.

    At the top, you will see the title Offer.

    Below that, all relevant trip details are listed.

    Accept Offer: If you want to accept the offer, simply click the Accept button. You will then receive a confirmation email that the ride has been booked.

    Clarify Questions: If you have questions or uncertainties about an offer, you can click the Contact button. This opens a chat window where you can communicate with the provider to determine whether the offer suits your needs.

    Report Suspicious Activity: If an offer seems suspicious or you notice unusual content, you can use the Report button in the top-right corner of the marker to report it. If a provider receives too many reports, the offer will be reviewed and the provider may be suspended to ensure user safety.
